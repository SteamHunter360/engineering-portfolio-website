Robotics Arm Control

A Python-based robotics simulator implementing forward kinematics, inverse kinematics, trajectory planning, workspace visualisation and interactive robot arm control.

View GitHub Repository Back to Projects

Project Overview

This project demonstrates core robotics concepts through a two-link planar robotic arm simulator. It was developed to strengthen practical understanding of robotics mathematics, Python programming, visualisation and engineering software development.

Key Features

Forward Kinematics Inverse Kinematics Interactive Sliders Click-to-Move Control Trajectory Planning Workspace Visualisation

Engineering Concepts

The project applies robotics principles such as calculating end-effector position from joint angles, solving joint angles from target coordinates, planning smooth movement between configurations and visualising the reachable workspace of the robot arm.

Technologies Used

Python NumPy Matplotlib Git GitHub

What I Learned

Through this project, I developed stronger practical understanding of robotics kinematics, engineering visualisation, interactive simulation and iterative software development using Git and GitHub.
