Predictive Maintenance Fault Detection

A Python-based engineering data project using vibration signals, FFT analysis, feature extraction and machine learning to classify normal and faulty machine behaviour.

View GitHub Repository Back to Projects

Project Overview

This project simulates a predictive maintenance workflow for rotating machinery. It generates synthetic vibration data, analyses frequency components using FFT, extracts diagnostic features and trains a machine learning classifier to identify faults.

Key Features

Synthetic Vibration Data FFT Analysis Fault Band Detection Feature Extraction Dataset Generation Random Forest Classifier

Engineering Concepts

The project applies signal processing, vibration analysis and data-driven fault detection. It demonstrates how changes in vibration frequency content can indicate early machine faults and how extracted features can be used for classification.

Technologies Used

Python NumPy Pandas Matplotlib Scikit-learn GitHub

What I Learned

Through this project, I developed practical experience in engineering data analysis, vibration signal processing, feature engineering and applying machine learning to mechanical condition monitoring problems.
